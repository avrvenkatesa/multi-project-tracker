Kanban Board

Visualize your workflow and manage work items with drag-and-drop simplicity.

1 Kanban Board Overview

The Kanban board provides a visual workflow representation with columns for each status:

To Do

Not started

In Progress

Active work

In Review

Under review

Resolved

Completed

Each column shows cards representing issues or action items in that status.

2 Accessing the Kanban Board

Open the Kanban board for any project:

  1. Navigate to your project
  2. Click the "Kanban" button in the project toolbar
  3. The board will display all issues and action items organized by status

Quick Access:

Use the Kanban button on the project card from the main page for direct access.

3 Card Information Display

Each Kanban card displays comprehensive information at a glance:

  • Title: Item name with truncation for long titles
  • Priority Badge: Color-coded badge showing priority level
  • Due Date: Color-coded date indicator (red for overdue, yellow for upcoming, green for ample time)
  • Assignee: Person responsible for the item
  • Creator: Shows who created the item with username
  • Attachment Count: Paperclip icon with number badge showing attached files

Creator Display:

The "Created by [Username]" footer on each card helps identify item ownership and provides accountability. This is especially useful when reviewing work or onboarding new team members.

Due Date Color Coding

Overdue

Past the due date - immediate attention needed

Due Soon

Due within 3 days - prioritize this work

On Track

More than 3 days away - on schedule

Attachment Badge:

When items have attachments, a paperclip icon with count badge appears on the card. This helps you quickly identify items with supporting documentation or evidence.

4 Advanced Multi-Criteria Sorting

Cards within each column can be sorted using multiple criteria to organize your work effectively:

Available Sort Options

Due Date Urgency (Default)

Automatically sorts by due date with color-coded urgency: Overdue → Due Soon → On Track

Refreshes automatically at midnight to keep urgency levels current

Priority Level

Sorts by priority: Critical → High → Medium → Low

Best for focusing on high-impact work first

Recently Updated

Shows most recently modified items first

Useful for tracking active work and recent changes

Manual Sort

Drag cards to arrange in custom order

Your custom order is preserved until you change sorting

Using Sort Controls

  1. Click the "Sort by" dropdown in the Kanban toolbar
  2. Select your preferred sorting method
  3. Cards will automatically rearrange in all columns
  4. The selected sort order is applied immediately

Automatic Daily Refresh:

When using "Due Date Urgency" sort, the Kanban board automatically refreshes at midnight to update color coding as items move from "On Track" to "Due Soon" or "Overdue".

5 Drag & Drop Updates

Update item status with simple drag and drop:

  1. Click and hold on any card
  2. Drag the card to the target column
  3. Release to drop the card
  4. The item's status automatically updates

Pro Tip:

Drag-and-drop is the fastest way to update status. No need to open the item detail form!

6 Filtering & Search

Focus on specific work items using filters:

  • Search: Filter by title or description keywords
  • Priority: Show only Critical, High, Medium, or Low priority items
  • Assignee: View items assigned to specific team members
  • Type: Filter to show only Issues or only Action Items

Filters apply across all columns simultaneously, making it easy to track specific work streams.

7 Swimlanes

Group cards horizontally for better organization:

Priority Swimlanes

Group by Critical, High, Medium, Low priority levels

Assignee Swimlanes

Group by team member for workload visibility

Category Swimlanes

Group by issue category or type

Toggle swimlanes using the view options in the board toolbar.

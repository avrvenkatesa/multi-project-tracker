Managing Projects

Learn how to create, configure, and organize your projects effectively.

1 Creating Projects

Projects are the top-level containers for your work:

  1. Click the "+ New Project" button from the main page
  2. Enter a descriptive project name
  3. Add a description explaining the project's purpose
  4. Select a template that matches your project type
  5. Click "Create Project"

Project Name Best Practices:

Use clear, specific names like "Website Redesign Q1 2025" instead of generic names like "Project 1"

2 Project Templates

Templates provide pre-configured categories and workflows:

Cloud Migration Template

Designed for infrastructure migration projects

Categories: Technical, Risk, Process, Compliance

Phases: Assessment, Planning, Migration, Validation

Software Development Template

Optimized for agile software projects

Categories: Bug, Feature, Enhancement, Technical Debt

Phases: Backlog, Development, Testing, Production

Infrastructure Template

For operations and DevOps teams

Categories: Incident, Maintenance, Upgrade, Security

Phases: Planning, Implementation, Testing, Deployment

Generic Template

Flexible for any project type

Categories: Task, Issue, Request, Other

Phases: To Do, In Progress, Review, Done

3 Viewing Projects

Access your projects from the main page:

  • My Projects: Shows all projects you're a member of
  • Project Card: Displays name, description, and member count
  • Quick Actions: View, Dashboard, Kanban, and Team buttons

Click any project card to view its details and work items.

4 Editing Project Settings

Project managers and admins can modify project settings:

  1. Open the project
  2. Click the "Settings" or gear icon
  3. Modify the name, description, or other settings
  4. Click "Save Changes"

Permission Required:

Only Project Managers and Admins can edit project settings. Team Members and Viewers have read-only access.

5 Archiving Projects

Archive completed projects to keep your workspace organized:

  1. Open the project settings
  2. Scroll to the "Archive Project" section
  3. Click "Archive This Project"
  4. Confirm the action

Important:

Archived projects are hidden from the main view but can be restored by administrators. All data is preserved.

File Attachments

Attach files, documents, screenshots, and other resources to issues and action items for better context and collaboration.

1 File Attachments Overview

The File Attachments feature allows you to:

  • Upload files: Attach documents, images, PDFs, spreadsheets, and more
  • View attachments: See all files attached to an issue or action item
  • Download files: Get local copies of attachments
  • Delete attachments: Remove files when no longer needed
  • Track context: Keep all relevant materials with the work item

Attachment Count Badges:

Items with attachments display a badge on Kanban cards showing the file count for quick reference.

2 Uploading Files to Items

To attach files to an issue or action item:

  1. Open the item by clicking on it in the Kanban board
  2. Navigate to the "Attachments" tab in the detail modal
  3. Click "Choose File" or drag and drop a file
  4. Select the file from your computer
  5. Click "Upload" to attach the file
  6. The file will appear in the attachments list

Supported File Types:

Most common file types are supported including PDFs, images (JPG, PNG, GIF), documents (DOC, DOCX), spreadsheets (XLS, XLSX), presentations (PPT, PPTX), and text files.

File Size Limits:

Maximum file size is typically 10MB per attachment. For larger files, consider using cloud storage links in comments.

3 Viewing Attachments

View all files attached to an item:

  • Open the issue or action item
  • Click the "Attachments" tab
  • See a list of all attached files with names and upload dates
  • The attachment count is also visible on the Kanban card

Attachment Badge on Cards:

Kanban cards show a paperclip icon with a number badge indicating how many files are attached. This helps you quickly identify items with supporting documentation.

4 Downloading Files

Download attachments to your local computer:

  1. Navigate to the Attachments tab
  2. Find the file you want to download
  3. Click the "Download" button (download icon)
  4. The file will be saved to your default downloads folder

Downloaded files retain their original filename for easy identification.

5 Deleting Attachments

Remove files that are no longer needed:

  1. Open the item and go to the Attachments tab
  2. Find the file you want to remove
  3. Click the "Delete" button (trash icon)
  4. Confirm the deletion when prompted
  5. The file is permanently removed

Project Dashboard

Monitor project health with real-time metrics, charts, and activity feeds.

1 Dashboard Overview

The project dashboard provides a comprehensive view of project status and health:

  • Real-Time Statistics: Current counts and percentages
  • Visual Charts: Status distribution and priority breakdown
  • Activity Trends: 30-day timeline showing project momentum
  • Activity Feed: Recent changes and updates
  • Team Metrics: Member workload and contributions

Access Dashboard:

Click "Dashboard" button on your project card or within the project view.

2 Key Metrics

The dashboard displays key performance indicators:

Total Issues

Count of all issues in the project

Total Action Items

Count of all action items

Completion Rate

Percentage of resolved items

Critical Priority

Count of high-urgency items

Team Size

Number of active members

Recent Activity

Items created in last 7 days

3 Charts & Visualizations

Interactive charts help visualize project data:

Issues by Status

Pie chart showing distribution across To Do, In Progress, In Review, and Resolved

Use to identify workflow bottlenecks

Issues by Priority

Pie chart displaying Critical, High, Medium, and Low priority breakdown

Use to focus team effort on high-priority work

Activity Trend (30 Days)

Line chart showing daily issue and action item creation

Use to track project momentum and identify activity patterns

All charts are powered by Chart.js and update in real-time as data changes.

4 Activity Feed

The activity feed shows recent project events:

  • New issues and action items created
  • Status changes on existing items
  • Assignment updates
  • Priority changes
  • Team member joins

Each entry includes timestamp, actor, action type, and affected item with a direct link.

Quick Navigation:

Click any activity item to jump directly to the related issue or action item.

5 Team Performance Metrics

View individual team member contributions:

Items Created

Number of issues and actions created by each member

Items Assigned

Current workload for each team member

Completion Rate

Percentage of assigned items marked as resolved

Use team metrics to balance workload and recognize top contributors.

Design Token Usage Guidelines

Documentation for future contributors to ensure consistent application of design tokens

Color Usage

Neutral 50-100: Backgrounds, borders, grid lines. Use Neutral 50 for Gantt grid backgrounds only.

Neutral 200: Default Gantt bars, card backgrounds on dashboard views.

Neutral 400: Progress bar fills, disabled states.

Neutral 700: Primary text color across all views.

Brand Primary: Buttons, accent bars, active states, links. WCAG AA compliant on white.

Brand Accent (#e8c555): Milestone indicators, icon fills, badges only. Avoid using for small text due to contrast limitations.

Semantic Colors: Success (completed), Warning (in-progress on Neutral 50+ backgrounds), Danger (critical path - muted tone), Info (notifications).

Typography Guidelines

3xl (28px): Page titles only. Use sparingly to avoid overwhelming modals.

2xl (24px): Section titles, modal headers.

xl (20px): Subsections, card headers.

base (16px): Main body content, descriptions, form labels.

sm (14px): Metadata, captions, helper text.

xs (12px): Table headers, timestamps, minimized content.

Gantt bars (10px): Task names on timeline bars only. Always use tooltips for truncated text exceeding bar width.

Spacing Application

xs (4px): Tight icon spacing, inline badge gaps.

sm (8px): Between related elements (label + input).

md (12px): Card padding, Gantt container padding.

lg (16px): Section gaps, column spacing.

xl (24px): Page padding, modal content spacing.

2xl (32px): Major section dividers.

3xl (48px): Hero sections, large page headers only.

Shadow & Accessibility

shadow-sm: Cards, dropdowns, subtle elevation.

shadow-md: Gantt bars, popovers, interactive elements.

shadow-lg: Modals, high-priority overlays.

shadow-focus: All keyboard-focusable elements. Must be visible on white backgrounds.

Contrast Requirements: All text must meet WCAG AA (4.5:1 for body text, 3:1 for large text). Test Brand Accent on backgrounds before implementation.

Colorblind Consideration: Critical vs Milestone bars should have additional visual cues (top border, pattern) beyond color alone.
